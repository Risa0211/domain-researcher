中古ドメインリサーチツール

ジャンルを選ぶだけで強い中古ドメインを自動リサーチ

ツール概要

ジャンルを選択するだけで、そのジャンルに関連する取得可能な中古ドメインを自動でリサーチするツールです。

  • ExpiredDomains.net から期限切れ・削除済みドメインを横断検索
  • Wayback Machine で過去のサイト内容を確認（日本語サイトだったか等）
  • Open PageRank API でドメインの評価スコアを取得
  • お名前.com / GoDaddy / Namecheap 等の取得先リンクを自動生成

全て無料で利用できます。

アクセス方法

ブラウザでアクセスするだけで使えます。インストール不要です。

https://domain-researcher.streamlit.app/

PC・スマホどちらからでもアクセスできます。

基本的な使い方

1
ジャンルを選択

プルダウンから登録済みジャンルを選択するか、自由入力欄にジャンル名を入力します。

登録済みジャンル：FX / 美容 / 転職 / ダイエット / 不動産 / クレジットカード / プログラミング / 恋愛・婚活 / 脱毛 / ガジェット / オンラインオリパ / トレカ

2
検索キーワードの確認・調整

ジャンルを選択すると、関連する英語キーワードがチェックボックスで一覧表示されます。

  • 除外したい場合 → チェックを外す
  • 追加したい場合 → カンマ区切りで入力
3
サイドバーでフィルター設定
項目説明
対象TLD.jp / .com / .net 等を選択
最低被リンク数指定数以上のBLを持つドメインのみ表示
最低ドメイン年齢指定年数以上の歴があるドメインのみ表示
Wayback Machine履歴ONで過去サイトの情報を取得
高評価ドメインも検索ドメイン名にキーワードを含まないが、過去にそのジャンルで運営されていた強いドメインも探す
ドメイン空き状況確認ONでWHOIS確認（件数分時間がかかります）
4
「リサーチ開始」ボタンを押す

全キーワードで横断検索が行われます。キーワード数に応じて数分かかる場合があります。

5
結果を確認

タブで切り替えて確認できます。

  • 全て 全検索結果
  • ドメイン名関連 ドメイン名にジャンルキーワードを含むもの
  • 高評価（過去運営関連） ドメイン名にキーワードはないが、過去にそのジャンルで運営されていたもの
  • .jp / .com / .net TLD別

各ドメインにお名前.com / GoDaddy / Namecheapのリンクが付いており、クリックで取得ページに飛べます。

6
ドメイン詳細確認・CSVダウンロード
  • ページ下部のプルダウンで個別ドメインの詳細を確認
  • 取得先ボタン（お名前.com / GoDaddy / Namecheap / Xserver / ムームー）から購入ページへ
  • 「CSVダウンロード」で結果を保存

2つの検索モードの違い

モード内容例（FXジャンル）
ドメイン名関連 ジャンルの関連キーワードがドメイン名に含まれるもの。メインの検索。 forex-guide.com
trading-info.net
高評価（過去運営関連） ドメイン名にキーワードは含まれないが、Wayback Machineで確認して過去にそのジャンルのサイトだった高BLドメイン。 example123.com
（過去タイトル: FX比較ガイド）
高評価モードの有効化：サイドバーの「高評価ドメインも検索」をONにしてください。Wayback Machineへのアクセスが増えるため、検索時間が長くなります。

コンテンツ判定キーワード（日本語）とは

「高評価ドメインも検索」をONにした場合のみ使用されます。

仕組み

  1. 高BL（被リンク数が多い）ドメインを取得
  2. Wayback Machineでそのドメインの過去のサイトタイトルを取得
  3. タイトルに日本語の判定キーワード（例: 「FX」「証券」「投資」）が含まれるかチェック
  4. 含まれていれば「過去にそのジャンルのサイトだった」と判定し、結果に追加
編集可能：キーワード確認画面の「過去コンテンツ判定キーワード」の折りたたみを開くと、チェックボックスで除外・追加ができます。

オプション設定

Open PageRank APIキー（無料）

設定するとPageRankスコアも結果に表示されます。

  1. https://www.domcop.com/openpagerank/auth/signup にアクセス
  2. 無料アカウントを作成
  3. 発行されたAPIキーをサイドバーの「Open PageRank APIキー」欄に入力

無料で1日100件まで取得可能です。

ExpiredDomains パスワード

入力しなくても検索できます。検索結果自体は変わりません。

ドメイン空き状況確認

ONにするとWHOISで各ドメインの登録状況を確認します。件数分の時間がかかるため、必要な時だけONにしてください。

ジャンル保存機能

自由入力でジャンルを入れ、キーワードを追加・除外した後に「このキーワード設定をジャンルとして保存」ボタンを押すと、そのキーワード設定が保存されます。

次回以降、プルダウンに表示されるようになります。

最終確認の推奨事項

このツールで候補を絞り込んだ後、以下の手動チェックを推奨します。

  1. Ahrefsでドメインを確認
    • DR（ドメインレーティング）の確認
    • 被リンクの質（スパムリンク比率）
    • 被リンク元サイトの関連性
  2. Wayback Machineで過去のサイトを目視確認
    • 結果テーブルの「Wayback」リンクをクリック
    • スパムサイトや低品質サイトでなかったか確認
  3. ドメインを取得
    • 結果テーブルのお名前.com / GoDaddy等のリンクから取得ページへ
重要：このツールはあくまで候補のリストアップが目的です。ドメインの最終的な価値判断は、必ずAhrefsやWayback Machineで手動確認してから行ってください。
