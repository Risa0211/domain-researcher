中古ドメインリサーチツール

使い方ガイド

1. ツール概要

ジャンルを選択するだけで、そのジャンルに関連する強い中古ドメインを自動リサーチするツールです。

以下の3つのサービスを組み合わせて、完全無料で利用できます。

  • ExpiredDomains.net — 期限切れドメインのデータベース
  • Wayback Machine — 過去のサイト内容を確認
  • Open PageRank API — ドメインのPageRankを取得

2. アクセス方法

Streamlit Community Cloudにデプロイ済みです。
ブラウザでアクセスするだけで使えます。インストール不要です。

アクセスURL: [デプロイ後に記入]

PC・スマホ・タブレットいずれのブラウザからも利用可能です。

3. 基本的な使い方

1 ジャンルを選択

プルダウンから登録済みジャンルを選択するか、自由入力でジャンルを指定します。

登録済みジャンル:

FX 美容 転職 ダイエット 不動産 クレジットカード プログラミング 恋愛・婚活 脱毛 ガジェット オンラインオリパ トレカ

上記にないジャンルも自由入力でOKです。

2 検索キーワードの確認・調整

選択したジャンルに応じて、ドメイン名検索用のキーワードが自動展開されます。

  • 不要なキーワードはチェックを外して除外
  • 追加したいキーワードはカンマ区切りで入力
3 サイドバーでフィルター設定

左のサイドバーで以下のフィルターを調整できます。

設定項目説明
対象TLD.jp / .com / .net 等から選択
最低被リンク数指定数以上のBLを持つドメインのみ表示
最低ドメイン年齢指定年数以上のドメインのみ表示
Wayback Machine履歴取得ON/OFF（ONで過去サイト情報を取得）
高評価ドメインも検索ON/OFF（ONで過去運営関連ドメインも検索）
4 リサーチ開始ボタンを押す

設定が完了したら「リサーチ開始」ボタンをクリックします。
検索には数十秒〜数分かかる場合があります。

5 結果の確認

結果は以下のタブに分かれて表示されます。

  • 全て — 全検索結果を一覧表示
  • ドメイン名関連 — キーワードがドメイン名に含まれるもの
  • 高評価（過去運営関連） — 過去にそのジャンルのサイトだったもの
  • TLD別 — TLDごとに分類して表示

各ドメインにはお名前.com / GoDaddy / Namecheapの取得リンクが付いています。
結果はCSVダウンロードも可能です。

6 ドメイン詳細確認

ページ下部のプルダウンで個別ドメインを選択すると、詳細情報を確認できます。

  • 被リンク数 / DP / ドメイン年齢
  • PageRank / アーカイブ数
  • 取得先ボタン: お名前.com / GoDaddy / Namecheap / Xserver / ムームー

4. 2つの検索モードの違い

モード内容
ドメイン名関連 ジャンルの関連キーワードがドメイン名に含まれるドメインを検索します。
メインの検索モードです。
高評価（過去運営関連） ドメイン名にキーワードは含まれないが、Wayback Machineで過去にそのジャンルのサイトだったことが確認できた高被リンクドメインです。
ポイント: 「高評価」モードは、ドメイン名だけでは見つけられない掘り出し物ドメインを発見できます。

5. コンテンツ判定キーワード（日本語）とは

「高評価ドメインも検索」をONにした場合にのみ使用される設定です。

Wayback Machineで過去サイトのタイトルタグを取得し、その中にこれらのキーワードが含まれるかどうかで「そのジャンルのサイトだったか」を自動判定します。

例: ジャンル「FX」の場合、コンテンツ判定キーワードに「FX, 為替, トレード」などが設定され、過去サイトのタイトルにこれらが含まれるドメインが「高評価」として表示されます。

6. オプション設定

Open PageRank APIキー

PageRankを表示するには、APIキーの入力が必要です。
以下のURLから無料で取得できます。

https://www.domcop.com/openpagerank/auth/signup

取得したAPIキーをサイドバーの入力欄に貼り付けてください。

ドメイン空き状況確認

ONにするとWHOISで空き状況を確認しますが、件数分の時間がかかります。

注意: ドメイン数が多い場合は確認に時間がかかるため、候補を絞り込んでからONにすることをおすすめします。

7. ジャンル保存機能

自由入力でジャンルを指定し、キーワードを調整した後、「このキーワード設定をジャンルとして保存」ボタンを押すと設定が永続化されます。

次回からプルダウンに表示されるようになるため、同じジャンルの再検索が簡単になります。

8. 最終確認の推奨事項

このツールで候補を絞り込んだ後、必ず手動チェックを行ってください。

推奨チェック項目:
  • Ahrefs で DR（ドメインレーティング）を確認
  • 被リンクの質を確認（スパムリンクが多くないか）
  • スパムスコアを確認
  • Wayback Machineのリンクから過去サイトの内容を目視確認
重要: ツールの数値だけで判断せず、過去のサイト内容や被リンクの質を総合的に判断してからドメインを取得してください。
